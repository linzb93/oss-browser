# 文件列表页面测试用例

## 1. 基础页面结构

- [ ] **页面加载**: 页面能正常加载，显示面包屑导航、工具栏按钮、文件列表。
- [ ] **面包屑导航**: 点击面包屑路径项，能正确跳转到对应目录；点击返回按钮能返回上一级；点击首页图标能回到首页。
- [ ] **拖拽上传区域**: 拖拽文件到页面时，显示提示区域；松开后区域消失。

## 2. 文件列表展示

- [ ] **文件列表加载**: 页面进入后能正确加载文件列表，显示文件名、大小、修改时间。
- [ ] **文件图标**: 目录显示文件夹图标，文件根据扩展名显示对应图标。
- [ ] **文件大小**: 文件大小显示格式化后的值（如 KB、MB）。
- [ ] **修改时间**: 目录显示“-”，文件显示格式化后的时间。
- [ ] **分页加载**: 滚动到底部时能加载更多文件。
- [ ] **预览图模式**: 在设置中开启预览图后，列表中显示图片缩略图。

## 3. 文件操作

- [ ] **进入目录**: 点击目录名称能进入该目录，面包屑更新。
- [ ] **文件点击**: 点击图片文件能在预览弹窗中打开；点击其他文件能下载。
- [ ] **获取地址**: 点击“获取地址”按钮，能将文件 URL 复制到剪贴板。
- [ ] **下载文件**: 点击“下载”按钮，能下载文件。
- [ ] **复制样式**: 当文件是图片且存在模板时，显示“复制样式”按钮，点击后能复制带样式的链接。
- [ ] **删除文件**: 点击删除按钮，弹出确认框，确认后能删除文件，列表刷新。

## 4. 批量操作

- [ ] **多选文件**: 勾选文件后，顶部出现“批量操作”下拉菜单。
- [ ] **批量复制地址**: 选择文件后点击“批量复制地址”，能将多个文件 URL 复制到剪贴板。
- [ ] **批量下载**: 选择文件后点击“批量下载”，能下载多个文件。
- [ ] **批量删除**: 选择文件后点击“批量删除”，弹出确认框，确认后能删除多个文件，列表刷新。
- [ ] **目录不可选**: 目录项的复选框为禁用状态。

## 5. 目录操作

- [ ] **创建目录**: 点击“创建目录”按钮，输入名称后能创建新目录，列表刷新。
- [ ] **同名目录**: 创建同名目录时，提示“存在同名目录，无需创建”。
- [ ] **空名称**: 创建目录时输入空名称，提示“请输入目录名称”。

## 6. 更多功能

- [ ] **设置**: 点击“更多功能”→“设置”，能打开设置弹窗。
- [ ] **查看收藏夹**: 点击“更多功能”→“查看收藏夹”，能打开收藏夹弹窗。
- [ ] **收藏**: 点击“更多功能”→“收藏”，能将当前路径添加到收藏夹。
- [ ] **设为首页**: 点击“更多功能”→“设为首页”，能将当前路径设为默认首页。
- [ ] **上传历史**: 点击“更多功能”→“上传历史”，能打开上传历史弹窗。
- [ ] **返回登录**: 点击“更多功能”→“返回登录”，能清除默认应用 ID 并返回登录页。

## 7. 文件上传

- [ ] **拖拽上传**: 拖拽文件到页面，能触发上传流程。
- [ ] **重名提示**: 上传重名文件时，弹出提示“下列文件已存在，是否覆盖？”。
- [ ] **覆盖上传**: 在重名提示中点击“覆盖”，能上传并覆盖原文件。
- [ ] **跳过上传**: 在重名提示中点击“不覆盖”，能跳过重名文件。
- [ ] **上传进度**: 上传过程中显示进度抽屉，展示上传文件列表。
- [ ] **上传完成**: 上传完成后，列表自动刷新。

## 8. 键盘快捷键

- [ ] **创建目录**: 按下快捷键能打开创建目录弹窗。
- [ ] **复制地址**: 选中文件后按下快捷键，能复制文件 URL。
- [ ] **上下导航**: 选中文件后，能用上下键切换选中项，列表自动滚动。

## 9. 异常处理

- [ ] **接口失败**: 文件列表加载失败时，提示“接口故障，请稍后再试”。
- [ ] **无效命令**: 输入无效命令时，提示“没有这个命令”。
- [ ] **无文件选择**: 进行批量操作时未选择文件，提示“请选择至少一个”。

## 10. 权限与数据

- [ ] **用户信息**: 页面顶部显示当前登录用户的域名信息。
- [ ] **默认路径**: 页面加载时自动跳转到用户设置的默认首页路径。
- [ ] **路径同步**: 面包屑路径与文件列表保持同步更新。
